AOCL LibM - Release Notes - AOCL version 4.2.0
----------------------------------------------

Highlights of AOCL LibM version 4.2.0
-------------------------------------
Added new vector array arithmetic functions:
 - vrsa_fmaxf, vrsa_fmaxfi, vrda_fmax and vrda_fmaxi
 - vrsa_fminf, vrsa_fminfi, vrda_fmin and vrda_fmini

Added new fast scalar functions:
 - pow
 - sin and sinf
 - cos and cosf
 - acosf

Optimized the performance of:
 - log
 - vrd8_exp2

Optimised the performance and improved the accuracy of:
 - tanhf, vr4s_tanhf, vr8s_tanhf and vr16s_tanhf
 - atan
 
Added optimized versions of fmax, fmaxf, fmod, fmodf, fdim and fdimf. 

Improved the examples.
